#+title: README

This is my personal configuration of Emacs. It divides into two
version, one is the most stable and probably would not breakup, the
other is the testing branch with much more features, which is I am
using. And if I some packages is stable enough, I will merge into the
master branch. Hope there is one day, I could settle down.

* Get Started
My configuration is constructed with org babel, so you couldn't not
find any ~.el~ file. You have to open the ~config.org~ file and press
~C-c C-v t~, which is a shortcut of ~org-babel-tangle~ command in the Emacs.

* Structure
My configuration mainly divided into 3 folders.

- ~lisp~: It contain all the files to tweak Emacs.
- ~site-lisp~: This folder contains the outside packages I added to
  Emacs which is configured by ~git submodule~.
- ~test-lisp~: This is the folder I test some new feature of Emacs. Or
  it can be the custom folder. Here I manage the package with
  ~git clone~.
